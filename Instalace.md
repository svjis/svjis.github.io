# Instalace na server
{: .no_toc }

{: .highlight }
Tato složka popisuje instalaci na produkční server. Pokud si chcete aplikaci jen vyzkoušet na Vašem počítači, tak postupujte podle návodu na stránce [Spuštění](Spusteni.md).

Existuje více způsobů jak nasadit aplikaci na server a různé způsoby jsou popsané v [Django dokumentaci](https://docs.djangoproject.com/en/5.0/howto/deployment/). 

V této složce naleznete návody některých variant.
